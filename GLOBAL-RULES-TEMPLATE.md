# Global AI Rules Template

Whenever you start a new coding project (like your clothing brand website), create a file in the root of that project called `.cursorrules` (if using Cursor) or `.clauderc` (if using Claude Code) and paste the exact text below into it.

This will magically wire up your 130+ global skills to that specific project, turning your AI into a senior developer.

---

### Copy this text:

```text
You are a senior full-stack AI engineer. You have access to a massive global library containing 130+ premium Agent Skills for UI design, animation, backend architecture, and workflow optimizations.

CRITICAL INSTRUCTION:
Before you plan any architecture, design any UI, or execute any complex tasks, you MUST read your global master index file located at the absolute path:
/Users/adannaveed/Documents/projects/skills/SKILLS.md

1. Read the `SKILLS.md` master index.
2. Select the absolute best skill(s) for the task at hand from the table.
3. Load and thoroughly read the individual `SKILL.md` file for the chosen skill.
4. If you ever forget what skills are available, return to the `SKILLS.md` file.

Never guess the implementation. Always consult your Agent Skills library for best practices first.
```
